06 — Brand Governance
AyuVinc master brand
Text only. Space Grotesk 300. CamelCase primary, uppercase for institutional contexts. English only — never with Devanagari. No symbol, mark, or icon in any context.
Module identity
All three modules (Dhara, Vaani, Setu) use identical typographic treatment. Layer number (01, 02, 03) is the only differentiator beyond the name. No color variation. No symbol variation.
Devanagari script
Always co-equal with English. Never a subtitle, caption, or translation. Minimum 16px on screen, 12pt in print. Optical size set to 96% of the paired English size. Noto Sans Devanagari weight 300 only.
Color usage
Teal (oklch 62% 0.10 195) for layer numbers and structural accents only. No per-module color differentiation. No gradient fills. No colored backgrounds for identity elements.
Alignment
All lockups are left-aligned. Never centered. Infrastructure reads as specification documents — structured, not designed. Centering introduces design register; left alignment asserts system.
Typography weight
Space Grotesk 300 for all display and identity use. Shifts to 400 below 24px for legibility. Never bolder than 400 in identity contexts. DM Sans 400 for all labels and descriptors.
What this system is not
Not startup branding. Not a product identity. Not an app icon system. Not a wellness or Ayurveda brand. Not multi-colored. Not symbolic. The identity communicates through restraint and system logic alone.
